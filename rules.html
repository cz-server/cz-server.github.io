Page 1/6 — What you need to know

  • 1. If you play on our server, you are responsible for knowing all the server rules.
  • 2. Ignorance of the rules does not exempt you from punishment — you are responsible for being aware of them.
  • 3. If someone hacks into your account and abuses your rank or breaks the rules, we are not responsible for that — you are!
  • 4. The server staff positions are: Police, Manager, Developer, CZ-Builder, Dep.Owner, Creator.
  • 5. Rank suspension is permanent, so be careful!

Page 2/6

  • 1.1 Spam, flood, insults, 18+ content, and excessive vulgar or offensive language in chat is forbidden.
    Punishment: /warn, /tempmute 30m, /tempmute 90m, /ban
  • 1.2 Begging in any form is forbidden.
    Punishment: /warn, /tempmute 30m, /tempmute 12h, /ban-cid
  • 1.3 Creating advertisements or promoting other servers is forbidden.
    Punishment: /ban-ip, /psban
  • 1.4 Using offensive language against the server is forbidden.
    Punishment: /warn, /tempmute 30m, /tempmute 90m, /psban

Page 3/6

  • 2.1 Use of game modifications, hacks, or third-party software is forbidden.
    Punishment: /tempban 48h, /ban, /ban-cid, /ban-ip
  • 2.2 Griefing after teleporting to a player, or killing them via /tp is forbidden.
    Punishment: /warn, /tempban 24h, /ban, rank withdrawal
  • 2.3 Using building commands in protected areas without the owner's permission is prohibited.
    Punishment: /ban, rank withdrawal
  • 2.4 Using Creative mode or Fly in the PvP arena is prohibited.
    Punishment: /kick, /tempban 24h, /ban
  • 2.5 Teleporting to a player without their permission (if the player asks you to leave the area) is forbidden.
    Punishment: /kick, /warn, /tempban 48h, /ban, rank withdrawal

Page 4/6

  • 3.1 Creating regions, account names, or clan names with vulgar or 18+ content is forbidden.
    Punishment: Warn in chat, /warn, /ban, /ban-cid
  • 3.2 Logging into accounts that do not belong to you, especially Administration accounts, is forbidden.
    Punishment: /kick, /ban-ip, /psban
  • 3.3 Building or placing signs with 18+ content in any form is forbidden.
    Punishment: /tempban 24h, /ban, /ban-cid
  • 3.4 Disrupting server functionality (creating lag) is forbidden.
    Punishment: /tempban 48h, /ban, /psban

Page 5/6

  • 4.1 Abusing commands such as /cc, /tell, or other similar commands is forbidden.
    Punishment: warn, /kick, /tempban 48h, /ban-ip, rank withdrawal
  • 4.2 Punishing players with non-existent or incorrect reasons is prohibited.
    Punishment: /tempban 24h, /tempban 7d, /ban, rank withdrawal
  • 4.3 AFK (away from keyboard) abuse / long AFK is managed (rules apply).
    Punishment: /kick 2/2, /tempban 24h, /ban

Page 6/6 — Notes

  • 1. If a player attempts to bypass a punishment or joins on another account and breaks the rules again, a /ban-ip or /ban-cid may be executed with the reason "Bypass".
  • 2. If you have been punished without a reason or with an invalid one, please contact us on Discord. It is necessary to have evidence! If you don't have any: create a ticket with /ticket.
  • 3. /psban is a command that can only be used by staff, and it permanently bans a player with no possibility of unban.
These are the server rules — follow them to keep the server safe and fun for everyone.
